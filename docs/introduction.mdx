## What It Is

Decision Engine is a Rust service that picks a gateway from an eligible list and records transaction outcomes for later routing decisions.

The repository currently ships:

- the routing API
- merchant and routing configuration APIs
- Docker Compose profiles for local bring-up
- Helm chart assets
- a React dashboard under `/decision-engine/` in production-style hosting
- Mintlify docs served in the dashboard profiles

## Main API Surface

The main endpoint groups are:

- `GET /health`, `GET /health/ready`, `GET /health/diagnostics`
- `POST /auth/signup`, `POST /auth/login`, `GET /auth/me`
- `POST /decide-gateway`
- `POST /update-gateway-score`
- `POST /merchant-account/create`
- `GET /merchant-account/{merchantId}`
- `DELETE /merchant-account/{merchantId}`
- `GET|POST /merchant-account/{merchantId}/debit-routing`
- `POST /routing/*`
- `POST /rule/*`
- `GET /analytics/*`

Use the OpenAPI-backed endpoint pages for request and response schemas.

## Run Locally

For API only:

```bash
docker compose up -d
curl http://localhost:8080/health
```

For API + dashboard + docs:

```bash
COMPOSE_PROFILES= docker compose --profile dashboard-postgres-ghcr up -d
```

URLs:

- API: `http://localhost:8080`
- Dashboard: `http://localhost:8081/decision-engine/`
- Docs: `http://localhost:8081/introduction`

Use [Local Setup](local-setup) for the full Compose, source-build, and Helm matrix.

## Dashboard Routes

When the dashboard profile is running, the React app is available at:

- `/decision-engine/`
- `/decision-engine/login`
- `/decision-engine/signup`
- `/decision-engine/onboarding`
- `/decision-engine/routing`
- `/decision-engine/routing/sr`
- `/decision-engine/routing/rules`
- `/decision-engine/routing/volume`
- `/decision-engine/routing/debit`
- `/decision-engine/decisions`
- `/decision-engine/analytics`
- `/decision-engine/audit`

For source-run Vite development, these routes are mounted at root, for example `http://localhost:5173/routing/rules`.
Source-run API calls use `/decision-engine-api/*`, which Vite proxies to `http://localhost:8080`.
Production builds call APIs through `https://app.hyperswitch.io/decision-engine/api/*` unless `VITE_API_BASE_PATH` overrides that base.

Use [Dashboard Guide](dashboard-guide) for route-by-route details.
