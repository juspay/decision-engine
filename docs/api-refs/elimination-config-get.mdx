# Get Elimination Config

## Request

```bash
curl -X POST "$BASE_URL/rule/get" \
  -H "Content-Type: application/json" \
  -d '{
    "merchant_id": "test_merchant_123423",
    "algorithm": "elimination"
  }'
```

## Response

```json
{
  "merchant_id": "test_merchant_123423",
  "config": {
    "type": "elimination",
    "data": {
      "threshold": 0.35
    }
  }
}
```
