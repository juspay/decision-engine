## Routing Approach Values

- `SR_SELECTION_V3_ROUTING`: choose the best eligible gateway from SR score at the active routing dimension.
- `SR_V3_DOWNTIME_ROUTING`: SR routing where one or more eligible gateways were deprioritized due to downtime.
- `SR_V3_ALL_DOWNTIME_ROUTING`: all eligible gateways are in downtime and the best degraded option is selected.
- `SR_V3_HEDGING`: exploration mode across eligible gateways regardless of SR ranking.
- `SR_V3_DOWNTIME_HEDGING`: hedging with some eligible gateways in downtime.
- `SR_V3_ALL_DOWNTIME_HEDGING`: hedging while all eligible gateways are in downtime.
